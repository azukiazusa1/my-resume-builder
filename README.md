# My Resume Builder

## Demo

### View storybook

https://azukiazusa1.github.io/my-resume-builder/

## Requirement

- Node.js 14.17.4
- Yarn 1.22.11

## Setup

### Install

```bash
yarn
```

### Run the development server

```bash
yarn dev
# or
yarn dev:mock # enable mock server by msw
```

### Test

```bash
yarn test:watch
```

### Lint

```bash
yarn lint:fix
yarn format
```

### Starts Storybook in development mode

```bash
yarn storybook
```
